Prosta aplikacja umożliwiająca:
- wklejenie textu
- posortowanie linii alfabetycznie
- usunięcie duplikatów
- pobieranie zaznaczonej linii do schowka

Wykożystywana przy zmianie uprawnień do zasobów użytkownika korporacyjnego , gdy lista obecnych uprawnień oraz lista uprawnień do zmiany była prawie identyczna, rózniąca sie pojedyńczymi sekcjami. 
Pozwalała oszczędzić czas i energię osoby zmieniającej uprawnienia.

</br>

A simple application that allows:
- pasting text
- sorting lines alphabetically
- removing duplicates
- saving the selected line to the clipboard

Used when changing permissions to corporate user resources, when the list of current permissions and the list of permissions to be changed were almost identical, differing only in individual sections.

It saved the time and energy of the person changing permissions.
